# Vite+ Template

`Vite` `React` `Vitest` `Tailwind` `Lucide` `Eslint (Common)` `Prettier (With the tailwind pluggin)`

### Create the project.

```
pnpm dlx degit pa-ku/vite-react-template.git app-name
```

### Access the project directory.

```
cd app-name``
```

### Install dependencies.

```
pnpm install
```
